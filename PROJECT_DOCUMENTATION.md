## Quick Portfolio Summary

### Credit Card Fraud Detection
Built an end-to-end fraud detection analysis using transaction data, exploratory data analysis, preprocessing, encoding, normalization, and multiple predictive models. The project compared Logistic Regression, Naive Bayes, SVM, Random Forest, Gradient Boosting Machine, K-Means, and MLP, with Gradient Boosting Machine emerging as the strongest model because of its high accuracy and balanced precision-recall performance.

### Hospitality Management System
Designed a database-driven hotel operations and analytics system to centralize booking management, guest data, room inventory, agent relationships, reporting, and business analytics. The project includes schema design, normalization, BCNF analysis, SQL analytics, indexing, and dashboard-style insights for hotel operations.

### Community Dreams Donor Management Application
Developed a donor management application using Streamlit, Python, MongoDB, PyMongo, and Pandas. The system supports CRUD operations, validation checks, duplicate detection, dashboard views, and Excel export to replace inconsistent spreadsheet-based donor tracking with a structured application workflow.
